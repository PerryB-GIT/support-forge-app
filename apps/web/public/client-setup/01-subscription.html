1. Subscription Setup

Set up your Claude Pro or Max subscription to unlock Claude Code access

Understanding Claude Plans

Claude Code requires an active Claude subscription. Here's what's available:

Plan Price Claude Code Access Best For
Free $0 No Testing Claude on web only
Pro $20/mo Yes Individual developers
Max (5x) $100/mo Yes Power users, more usage
Max (20x) $200/mo Yes Heavy daily use
Team Premium $150/seat/mo Yes Teams, shared billing

Our Recommendation

Start with Claude Pro ($20/mo) - It provides full Claude Code access and is sufficient for most individual developers. Upgrade to Max if you hit usage limits frequently.

Step-by-Step: Create Your Account

1

Go to Claude.ai

Visit https://claude.ai in your browser
2

Click "Sign Up" or "Get Started"

You'll see this button in the top right corner
3

Create your account

Options: Email + password, Google account, or Apple account
4

Verify your email

Check your inbox for the verification link
5

Upgrade to Pro or Max

Click your profile icon → "Upgrade to Pro" or visit Settings → Subscription
6

Enter payment information

Credit card or debit card accepted
7

Confirmation

You'll receive an email confirming your subscription is active

Verify Claude Code Access

Once subscribed, verify your Claude Code access:

  1. Go to claude.ai/settings
  2. Look for "Claude Code" or "CLI Access" section
  3. It should show as "Enabled" or "Available"
Note: Claude Code access is automatically included with Pro, Max, and Team Premium plans. No additional setup is needed on the Anthropic side.

Troubleshooting

"Claude Code not available"

  • Ensure your subscription is active (check Settings → Subscription)
  • Try logging out and back in
  • Contact Anthropic support if issues persist

Payment declined

  • Verify card details are correct
  • Check with your bank for international transaction blocks
  • Try a different payment method

Can't find upgrade option

  • Click your profile picture in the top right
  • Select "Settings" from the dropdown
  • Navigate to "Subscription" or "Billing"
