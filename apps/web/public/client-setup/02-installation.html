2. Installation Guide

Install all prerequisites and Claude Code CLI on Windows

Quick Option: Download our automated installer to install everything with one click.

Prerequisites Overview

Component Required? Purpose
Git for Windows Required Version control, Git Bash terminal
Node.js LTS Required JavaScript runtime for development
Python 3 Optional Python development, some tools
WSL 2 Optional Linux environment on Windows
Docker Desktop Optional Containerization, local dev environments

Manual Installation Steps

1. Git for Windows

Required
  1. Download from git-scm.com/download/windows
  2. Run the installer
  3. Accept default settings (or customize as needed)
  4. Ensure "Git from the command line" is selected
  5. Complete installation

Verify installation:

git --version

Expected: git version 2.47.x or higher

2. Node.js LTS

Required
  1. Download from nodejs.org (choose LTS version)
  2. Run the MSI installer
  3. Accept the license agreement
  4. Use default installation path
  5. Ensure "Add to PATH" is checked
  6. Complete installation

Verify installation:

node --version npm --version

Expected: v22.x.x for Node, 10.x.x for npm

3. Python 3 (Optional)

Optional
  1. Download from python.org/downloads
  2. Run the installer
  3. Important: Check "Add Python to PATH"
  4. Click "Install Now"
  5. Complete installation

Verify installation:

python --version

Expected: Python 3.12.x or higher

4. WSL 2 (Optional)

Optional

Open PowerShell as Administrator and run:

wsl --install
Note: Restart your computer after installation to complete WSL setup.

Verify installation (after restart):

wsl --version

5. Docker Desktop (Optional)

Optional
  1. Download from docker.com/products/docker-desktop
  2. Run the installer
  3. Accept the license
  4. Choose WSL 2 backend (recommended)
  5. Complete installation
  6. Launch Docker Desktop and complete first-run setup

Verify installation:

docker --version

Install Claude Code CLI

With prerequisites installed, install Claude Code:

Option A: Using CMD / Git CMD

curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd

Option B: Using PowerShell

irm https://claude.ai/install.ps1 | iex

Option C: Using WinGet

winget install Anthropic.Claude
After installation: Close and reopen your terminal, then run claude --version to verify.

Final Verification

Run the health check to ensure everything is working:

claude doctor

Expected output:

Checking environment... ✓ Operating system: Windows 10/11 ✓ Shell: cmd.exe / PowerShell ✓ Git: 2.47.x ✓ Node.js: 22.x.x ✓ Authentication: Not logged in All checks passed!
