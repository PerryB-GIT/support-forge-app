3. Git CMD Installation

Step-by-step guide using Windows Command Prompt with Git

Open Git CMD

1

Search for Git CMD

Press Windows + S and type Git CMD

2

Run as Administrator

Right-click on "Git CMD" and select "Run as administrator"

You should see a command prompt:

C:\Users\YourName>

Verify Git Installation

git --version

Expected output: git version 2.47.1.windows.2 or similar

If you see an error: Git isn't installed properly. Download it from git-scm.com

Install Claude Code

3

Run the Installer

Copy and paste this command:

curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd

What this does:

  1. Downloads the installer script from claude.ai
  2. Runs the installer
  3. Cleans up the temporary file

Expected output:

Downloading Claude Code... Installing Claude Code... Claude Code installed successfully! Installation complete. Run 'claude' to get started.

Verify Installation

4

Close and Reopen Git CMD

This refreshes your PATH so the claude command is recognized

5

Check Version

claude --version

Expected: claude-code version X.X.X

6

Run Health Check

claude doctor

Authenticate

7

Navigate to Your Project

cd C:\Users\YourName\Projects\my-project

Or create a test folder:

mkdir C:\ClaudeTest cd C:\ClaudeTest
8

Start Claude Code

claude
9

Complete Authentication

  1. Browser opens automatically with Anthropic login
  2. Sign in with your Claude Pro/Max credentials
  3. Click "Allow" to authorize Claude Code
  4. Return to terminal - you're ready!
Success! You should see: "Authentication successful! Welcome to Claude Code."

Common Commands

Command Description
claude Start Claude Code in current directory
claude "do something" Start with a specific task
claude --version Check installed version
claude doctor Run health check
claude update Update to latest version
claude logout Sign out

Troubleshooting

"curl is not recognized"

Git CMD should include curl. If not, reinstall Git for Windows with all components.

"claude is not recognized" after installation

set PATH=%PATH%;%USERPROFILE%\.claude\bin

Then try claude --version again.

"Authentication failed"

  1. Ensure your subscription is active at claude.ai
  2. Try: claude logout then claude
  3. Check if firewall is blocking claude.ai
