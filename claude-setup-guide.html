Step 1: Install Prerequisites

Set up the required tools before installing Claude enhancements

1

Download & Install Node.js

Required for NPX-based MCP servers. Download from nodejs.org

# Verify installation node --version npm --version
2

Download & Install Python 3.10+

Required for Python-based MCP servers. Download from python.org

# Verify installation python --version # Install UV package manager (required for some extensions) pip install uv
3

Install Git

Required for cloning MCP server repositories. Download from git-scm.com

4

Download Claude Desktop

Download and install Claude Desktop from claude.ai/download

Pro Tip Make sure to add Python and Node.js to your system PATH during installation for easier command-line access.

Step 4: Add Plugin Marketplaces

Extend Claude with community plugins

1

Add Superpowers Marketplace

Run this command in Claude Code to add the marketplace:

/plugin marketplace add obra/superpowers-marketplace
2

Add Anthropic Agent Skills

Add the official skills marketplace:

/plugin marketplace add anthropics/skills
3

Install Plugins

Install the recommended plugins:

# Superpowers (TDD, debugging, workflows) /plugin install superpowers@superpowers-marketplace # Document Skills (PDF, DOCX, PPTX, XLSX) /plugin install document-skills@anthropic-agent-skills
