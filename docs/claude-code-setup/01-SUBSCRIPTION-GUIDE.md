**Support Forge Client Resource**
*Getting Started with Claude Pro or Max*

## Overview

Claude Code is Anthropic's official AI-powered coding assistant that runs directly in your terminal. To use Claude Code, you'll need an active Claude subscription that includes CLI access.

## Available Plans

### For Individuals

| Plan | Price | Claude Code Access | Best For |
|------|-------|-------------------|----------|
| **Claude Pro** | $20/month ($17/month annually) | Yes - Standard limits | Personal projects, learning |
| **Claude Max** | $100/month | Yes - 5x more capacity | Heavy daily usage |
| **Claude Max+** | $200/month | Yes - 20x more capacity | Power users, large codebases |

### For Teams

| Plan | Price | Claude Code Access | Best For |
|------|-------|-------------------|----------|
| **Standard Seat** | $25/person/month | No | Web interface only |
| **Premium Seat** | $150/person/month | Yes | Development teams |
| **Enterprise** | Custom pricing | Yes | Large organizations |

> **Recommendation**: Most individuals should start with **Claude Pro** at $20/month. Upgrade to Max if you hit usage limits frequently.

## Step-by-Step: Setting Up Your Subscription

### Option A: Claude Pro/Max (Recommended for Individuals)

1. **Visit the pricing page**
   - Navigate to: https://claude.ai/pricing

2. **Select your plan**
   - Click "Subscribe" under Claude Pro or Claude Max
   - Choose monthly or annual billing (annual saves ~15%)

3. **Create or sign in to your account**
   - Use email + password, or
   - Sign in with Google
   - Complete email verification if new account

4. **Enter payment information**
   - Credit/debit card required
   - Subscription begins immediately

5. **Confirm subscription**
   - You'll receive a confirmation email
   - Your account is now ready for Claude Code

### Option B: Claude for Teams (For Organizations)

1. **Visit team pricing**
   - Navigate to: https://claude.com/pricing
   - Scroll to "Claude for Teams"

2. **Choose seat types**
   - Minimum 5 team members
   - Premium Seats ($150/person) include Claude Code
   - Standard Seats ($25/person) do NOT include Claude Code

3. **Complete team setup**
   - Add team member emails
   - Assign seat types
   - Configure billing (annual recommended for discount)

4. **Team members authenticate**
   - Each member receives invite email
   - Sign in with team credentials
   - Claude Code access enabled for Premium seats

## Verifying Your Subscription

After subscribing, verify your plan includes Claude Code:

1. Sign in at https://claude.ai
2. Click your profile icon (top right)
3. Select "Settings" → "Subscription"
4. Confirm you see "Pro", "Max", or "Premium" status

## Managing Your Subscription

### Upgrade/Downgrade
- Settings → Subscription → Change Plan

### Cancel Subscription
- Settings → Subscription → Cancel
- Access continues until end of billing period

### Billing History
- Settings → Subscription → Billing History
- Download invoices for expense reports

## Usage Limits & Monitoring

### Understanding Limits

Claude Code usage counts against your subscription limits:

| Plan | Approximate Message Limit |
|------|---------------------------|
| Pro | ~100-200 complex coding tasks/month |
| Max (5x) | ~500-1000 complex tasks/month |
| Max (20x) | ~2000-4000 complex tasks/month |

### Checking Usage
- Visit https://claude.ai/settings/usage
- View daily and monthly consumption
- Receive warning at 80% capacity

### If You Hit Limits
1. Wait for daily reset (midnight UTC)
2. Upgrade to higher tier
3. Use API credits as backup (Console billing)

## Troubleshooting

### "Subscription Not Found" Error
- Ensure you're signed into the correct account
- Check that payment processed successfully
- Contact support@anthropic.com

### "Claude Code Not Available" on Team Plan
- Verify you have a **Premium Seat** (not Standard)
- Ask your team admin to upgrade your seat type

### Payment Failed
- Update card details in Settings → Billing
- Try a different payment method
- Check for bank blocks on international transactions
