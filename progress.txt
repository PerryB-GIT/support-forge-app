## Iteration 1 - US-001

**Completed:** Added CSS animation utilities and hover effects to globals.css

**Learnings:**
- Project uses Tailwind CSS v4 with @import "tailwindcss" syntax
- CSS custom properties already defined for brand colors (--accent-glow, --forge-purple, etc.)
- No dedicated typecheck script - use `npx tsc --noEmit` in apps/web directory
- Added prefers-reduced-motion support for accessibility

**CSS Classes Added:**
- `.card-hover` - lift + glow effect on hover
- `.glow-purple` - purple glow on hover
- `.scale-hover` / `.scale-hover-subtle` - scale effects
- `.animate-fade-up` / `.animate-fade-up-stagger` - scroll reveal
- `.animate-pulse-glow` - pulsing glow effect
- `.animate-float` - floating animation

**Files changed:**
- apps/web/src/app/globals.css

## Iteration 2 - US-002

**Completed:** Added gradient background system to globals.css

**Learnings:**
- CSS pseudo-elements (::before, ::after) work well for overlay effects
- SVG data URIs inline in CSS work for noise textures (keep opacity very low ~0.03)
- Need `pointer-events: none` on overlay pseudo-elements to allow clicks through
- Grid patterns use repeating linear-gradient with background-size for tiling
- Glow orbs use radial-gradient with blur filter for soft light effect

**CSS Classes Added:**
- `.hero-gradient` - main hero linear-gradient with purple spotlight ::before
- `.hero-spotlight-right` - additional right-side radial gradient ::after
- `.section-gradient` - vertical gradient for content sections
- `.surface-gradient` - diagonal gradient for cards/elevated areas
- `.noise-texture` - subtle grain effect using SVG feTurbulence
- `.grid-pattern` - tech aesthetic grid overlay
- `.glow-orb` / `.glow-orb-small/medium/large` - decorative blur effects

**Files changed:**
- apps/web/src/app/globals.css
